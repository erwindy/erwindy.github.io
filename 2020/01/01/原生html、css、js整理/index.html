onload和onready比较：

window.onload() 必须等到网页全部加载完毕（包括图片），然后执行js代码。
只执行一次，后面的会把前面的覆盖掉。
body.onload()与windows.onload()都是所有元素加载完成后执行，同时出现，只会执行一个（后面会覆盖前面）
$(document).ready() 只需等待网页中的DOM结构加载完毕，就能执行js代码。
可以执行多次，不会覆盖。
简写为$(function(){})
hasPrototypeProperty(obj, name)用于判断name是否在obj中（原型或者实例）
obj.hasOwnProperty(name)判断name是否是obj的实例属性或者方法。
for i in obj 遍历obj中所有可以枚举的属性（symbol不可枚举）
name in obj 判断name是否在obj(原型和实例)中。
delete obj.name 删除属性。
